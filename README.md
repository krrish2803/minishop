# E-Commerce Backend System

## 📌 Features
- Product CRUD
- Customer Management
- Order Management
- Aggregation (Sales Insights)

## ⚙️ Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)

## 🚀 API Endpoints

### Products
- POST /api/products
- GET /api/products
- PUT /api/products/:id
- DELETE /api/products/:id

### Customers
- POST /api/customers
- GET /api/customers

### Orders
- POST /api/orders
- GET /api/orders
- GET /api/orders/sales
- GET /api/orders/top-product

## ▶️ Run Locally

```bash
npm install
npm run dev
